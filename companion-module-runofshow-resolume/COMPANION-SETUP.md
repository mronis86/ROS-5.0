# Companion v3 – Resolume test module

Same rules as the main module: **Developer → Module path** must be the **parent** of `companion-module-runofshow-resolume`.

Example:

```text
C:\Users\audre\OneDrive\Desktop\ROS-5.0
```

Both `companion-module-runofshow` and `companion-module-runofshow-resolume` can coexist.

```bash
cd companion-module-runofshow-resolume
npm install
```

Restart Companion. Add a new connection: **Run of Show (Resolume)**.

See `README.md` in this folder for Resolume OSC routing.
