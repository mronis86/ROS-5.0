ROS LED Spout

Live LED page → Spout, or offline lossless animation pack → Spout

Offline backup workflow

  1. Live (normal): Source = Live page → Start Spout (cloud).
  2. Before show / for safety: fill Live fields → choose format → Bake pack….
  3. If cloud is down: Source = Offline prerender → point at that pack → Start Spout → run cues in offline-show (:3004).
Bake formats (select one or more) Multiple selections create separate format folders from one frame capture.

Bake pack… records each cue’s animate-in in the selected format. Offline Spout plays that clip once when you load/run the cue. Default output and bake rate is 60 FPS; enable the checkbox to use the selected rate.
